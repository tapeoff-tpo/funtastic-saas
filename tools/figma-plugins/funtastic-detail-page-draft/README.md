# Funtastic AI Detail Page Draft plugin

이 플러그인은 SaaS의 `Figma 초안 제작 요청` 작업을 현재 Figma 파일 안에 편집 가능한 상세페이지 프레임으로 만듭니다.

1. Figma Desktop에서 `Plugins > Development > Import plugin from manifest...`를 선택합니다.
2. 이 폴더의 `manifest.json`을 선택합니다.
3. `ai 생성 상세페이지` 파일을 열고 `Plugins > Development > Funtastic AI Detail Page Draft`를 실행합니다.
4. SaaS 상세페이지 제작 화면에서 `Figma 연결 코드 만들기`를 누르고 코드를 복사합니다.
5. 플러그인에 코드를 입력해 연결한 뒤 `대기 작업 전부 제작`을 누릅니다.

플러그인은 새 Figma 파일이나 프로젝트를 만들지 않습니다. `AI 상세페이지` 페이지에 상품별 초안을 추가하고 SaaS 작업을 `검수 필요` 상태로 바꿉니다.
