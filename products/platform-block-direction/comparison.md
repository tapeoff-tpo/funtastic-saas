# Direction Comparison

No direction is selected in this phase. The comparison identifies the decision the user must make before CAD.

| Dimension | A: Soft Cell | B: Facet Totem | C: Frame & Core |
| --- | --- | --- | --- |
| First impression | Calm, friendly block collection | Graphic, upright collectible blocks | Architectural living-system grid |
| Block proportion | Low and broad | Tall with mixed heights | Colored core inside visible frame |
| Orientation cue | Cell and front grip relief | Clipped corner | Frame opening and core perimeter |
| Push action | Broad block into broad cell | Tall block into shallow cell | Core directly into open frame |
| Pull action | Front relief plus exposed perimeter | Tall side-face grip | Exposed core rim |
| Horizontal growth | Strips and 2x2 trays | Rows of alternating heights | Continuous frame lattice |
| Vertical growth | Direct visual stacking | Totem-like stacking | Stepped frame hierarchy |
| Strongest value | Immediate and interior-friendly | Collection identity and color impact | Platform longevity and visible capacity |
| Main risk | Can resemble a premium button/dock | Can become toy-like or obstructive | Can resemble an organizer rather than blocks |
| Example modules | Tray, cup, cable | Cup, phone rest, cable | Tray, cup, cable |

## Common hard-gate result

All three directions intentionally show:

- A graspable block body
- Straight push-in and pull-out interaction
- 2x2 and horizontal expansion
- A vertical or hierarchical expansion cue
- Multiple living-product functions in one family
- Pastel and vivid block combinations
- No stud field, brick-copy proportion, slide, hook-and-slide, or twist-lock action

## User decision required

Choose the product-family attitude before engineering:

- **A** for the calmest, broadest desk-interior fit.
- **B** for the strongest collectible block identity.
- **C** for the clearest long-lived frame and replaceable-module architecture.

The selection is a direction approval, not approval of the depicted internal connection.
