# Funtastic 1688 URL Collector

## 설치

1. 이 폴더를 로컬에 둡니다.
2. Chrome에서 `chrome://extensions`를 엽니다.
3. 개발자 모드를 켜고 `압축해제된 확장 프로그램을 로드합니다`를 선택합니다.
4. 이 폴더를 지정한 뒤 Funtastic SaaS 품목 화면을 새로고침합니다.

## 동작

- SaaS가 주문번호가 `3`으로 시작하고 구매 URL이 비어 있는 발주건만 전달합니다.
- 확장프로그램은 로그인된 1688 주문조회 화면을 주문번호별로 순회합니다.
- 수집 결과 저장과 최종 매칭은 SaaS 서버에서 다시 검증합니다.
- 기존 구매 URL은 덮어쓰지 않습니다.
