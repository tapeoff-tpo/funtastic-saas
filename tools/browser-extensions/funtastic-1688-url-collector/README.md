# Funtastic 1688 URL Collector

## 설치

1. 이 폴더를 로컬에 둡니다.
2. Chrome은 `chrome://extensions`, Edge는 `edge://extensions`를 엽니다.
3. 개발자 모드를 켜고 `압축해제된 확장 프로그램을 로드합니다`를 선택합니다.
4. 이 폴더를 지정한 뒤 Funtastic SaaS 품목 화면을 새로고침합니다.

## 동작

- 폴더에 `order-queue.json`이 있으면 네 개 발주 엑셀에서 만든 주문번호 전체를 우선 사용합니다.
- 주문번호 접두어로 1688 계정을 추측하지 않고 현재 로그인된 계정에서 모든 번호를 순서대로 조회합니다.
- 확장프로그램은 로그인된 1688 주문조회 화면을 주문번호별로 순회하고 상품 영역을 열어 상세주소를 확인합니다.
- 각 결과가 SaaS에 저장됐다는 확인을 받은 뒤 다음 주문으로 이동합니다.
- 중단하거나 브라우저를 다시 열어도 최대 최근 9건만 재확인하고 저장 지점부터 이어갑니다.
- 미발견·복수 링크·저장 오류는 전체 순회가 끝날 때까지 누적합니다.
- 기존 구매 URL은 덮어쓰지 않습니다.

## URL 검증

- 품목 화면의 `URL 검증`은 이미 등록된 1688 구매 URL을 실제 로그인된 1688 화면에서 하나씩 엽니다.
- 결과는 `정상 열림`, `상품 없음`, `확인 필요`로 나뉩니다. 로그인 또는 1688 보안 확인이 나오면 자동으로 우회하지 않고 작업을 멈춥니다.
- 중단하거나 브라우저를 다시 열어도 마지막 확인 지점부터 이어집니다. URL 자체나 품목 정보는 변경하지 않습니다.
